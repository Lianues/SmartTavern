# Portrait Background Images (竖版背景图片)

This folder contains background images optimized for **mobile phones and portrait tablets** (portrait orientation).

存放**手机和竖屏平板**使用的竖版背景图片。

## Required Files / 所需文件

| Filename | Description |
|----------|-------------|
| `HomePage.png` | Start page background / 开始页背景 |
| `ThreadedChat.png` | Threaded chat page background / 楼层对话页背景 |
| `SandboxChat.png` | Sandbox page background / 沙盒页背景 |

## Recommended Dimensions / 推荐尺寸

- **Aspect Ratio**: 9:16 or 9:19.5 (for modern phones)
- **Resolution**: 1080x1920 or higher
- **Format**: PNG (preferred) or JPG

---

**Note**: The system will automatically select portrait images when the viewport height is greater than width.

# Landscape Background Images (横版背景图片)

This folder contains background images optimized for **desktop and tablet** (landscape orientation).

存放**桌面端和平板电脑**使用的横版背景图片。

## Required Files / 所需文件

| Filename | Description |
|----------|-------------|
| `HomePage.png` | Start page background / 开始页背景 |
| `ThreadedChat.png` | Threaded chat page background / 楼层对话页背景 |
| `SandboxChat.png` | Sandbox page background / 沙盒页背景 |

## Recommended Dimensions / 推荐尺寸

- **Aspect Ratio**: 16:9 or 16:10
- **Resolution**: 1920x1080 or higher
- **Format**: PNG (preferred) or JPG

---

**Note**: The system will automatically select landscape images when the viewport width is greater than height.
